## 📖 Overview

WanderWise is a comprehensive, modern web application designed to revolutionize how you plan and organize your travels. Built with cutting-edge web technologies, it combines beautiful design with powerful functionality to create the ultimate travel planning experience.

Whether you're planning a weekend getaway or a month-long adventure, WanderWise provides all the tools you need in one elegant, user-friendly interface.

## ✨ Key Features

### 🎯 Core Functionality
- **📅 Interactive Itinerary Planner** - Create detailed day-by-day schedules with drag-and-drop simplicity
- **🗺️ Smart Route Finder** - Get optimized transportation recommendations with real-time suggestions
- **✅ Preparation Checklist** - Never forget important tasks with our comprehensive pre-trip checklist
- **🧳 Intelligent Packing Assistant** - Receive customized packing lists based on destination, weather, and trip type
- **📄 Document Tracker** - Organize travel documents with expiry date alerts and reminders
- **💱 Currency Converter** - Real-time currency conversion with historical rate tracking

### 🎨 User Experience
- **🌎 Interactive Destination Gallery** - Explore beautiful destinations with high-quality imagery
- **🔍 Advanced Search & Filtering** - Find specific locations with smart search functionality
- **🗺️ Regional Organization** - Browse destinations by continent (Europe, Asia, Americas, Africa, Oceania)
- **🌓 Dark/Light Mode** - Seamlessly switch between themes for optimal viewing
- **📱 Mobile-First Design** - Fully responsive across all devices and screen sizes
- **🔄 Back to Top Navigation** - Smooth scrolling with convenient navigation

### 🎪 Design Excellence
- **✨ 3D Hover Effects** - Engaging card rotations and interactive animations
- **📊 Modern Statistics Cards** - Beautiful data visualization with smooth transitions
- **🎭 Smooth Animations** - Polished micro-interactions throughout the interface

## 🚀 Quick Start

### Prerequisites
- Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- No additional software installation required!

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/loopholend/WanderWise-Smart-routes.-Hidden-gems.-Travel-smarter.-.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd WanderWise-Smart-routes.-Hidden-gems.-Travel-smarter.-
   ```

3. **Launch the application**
   ```bash
   # Simply open index.html in your preferred browser
   # Or use a local server for optimal performance:
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

## 📱 How to Use

### 📅 **Itinerary Planning**
1. Click on the "Itinerary" tab
2. Add destinations, dates, and planned activities
3. Track your trip statistics (duration, activities, budget)
4. View your complete itinerary at a glance

### 🗺️ **Smart Route Planning**
1. Navigate to the "Smart Routes" tab
2. Enter your starting point and destination
3. Receive intelligent transportation recommendations
4. View estimated travel times and costs

### ✅ **Preparation Management**
1. Access the "Prep Checklist" tab
2. Complete essential pre-trip tasks
3. Track your progress with visual indicators
4. Never miss important preparations again

### 🧳 **Packing Optimization**
1. Open the "Packing" tab
2. Select your trip type, duration, and expected weather
3. Receive personalized packing recommendations
4. Check items off as you pack

### 📄 **Document Organization**
1. Visit the "Documents" tab
2. Add important travel documents
3. Set expiry date reminders
4. Keep all documents organized in one place

### 💱 **Currency Management**
1. Use the "Currency" tab
2. Convert between INR and international currencies
3. View current exchange rates
4. Plan your budget effectively

### 🌎 **Destination Discovery**
1. Explore the "Gallery" tab
2. Browse stunning destination imagery
3. Filter by regions and interests
4. Discover new places to visit
